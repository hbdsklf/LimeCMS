Eine Tag Cloud ist eine Methode zur Informationsvisualisierung, bei der eine Liste aus Schlagworten zweidimensional alphabetisch sortiert angezeigt wird, wobei einzelne unterschiedlich gewichtete Worte grösser oder auf andere Weise hervorgehoben dargestellt werden. Wortwolken werden zunehmend beim gemeinschaftlichen Indexieren und in Blogs eingesetzt. [Quelle]
